Level 2: Pause

Das System soll nach einer fünfsekündigen Pause melden, dass es bereit ist.

Pausiere das Skript

Damit Python Pausen einlegen kann, lädst du ganz am Anfang mit import timeLädt die Zeitfunktionen in dein Programm. das Modul time. Mit time.sleep(5)Pausiert den Code für 5 Sekunden. wartet der Computer genau fünf Sekunden.

lade Werkzeugkiste time import time
zeige Text "Verbindung wird hergestellt..." print("Verbindung wird hergestellt...")
pausiere (s) 5 time.sleep(5)
print("System bereit!") print("System bereit!")

Python-Codeausschnitt:

import time
print("Verbindung wird hergestellt...")
time.sleep(5)

Dein Schritt

Kopiere den Python-Codeausschnitt von oben in das Feld Python-Code. Ergänze danach print("System bereit!") und führe den vollständigen Code aus.

Erklärung

Das Python-Modul time zählt in Sekunden. time.sleep(5) pausiert das Programm fünf Sekunden. Erst danach zeigt print("System bereit!") die Bereitschaft im Ergebnis an.
