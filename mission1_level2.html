Level 2: Einen Hack simulieren

Wir simulieren jetzt eine kurze Wartepause, bevor das System antwortet.

Pausiere das Skript

Damit Python Pausen einlegen kann, musst du ganz am Anfang mit import timeLädt zusätzliche Fähigkeiten in dein Programm, hier die Zeit-Funktionen. das Modul time laden. Mit time.sleep(1)Pausiert den Code für 1 Sekunde. wartet der Computer dann genau eine Sekunde.

pausiere (ms) 1000

In Python (1000ms = 1 Sekunde):

time.sleep(1)

Aufgabe

1. Lade ganz oben das Modul: import time
2. Mache einen print() mit "Verbindung wird hergestellt...".
3. Mache eine Pause mit: time.sleep(1)
4. Klicke "Code Ausführen".
