Willkommen am Terminal, Agent! Dein Code hat in der Vergangenheit aus vielen bunten Blöcken bestanden. Aber als echter Hacker tippen wir unsere Befehle nun direkt in den Computer.

## 🐍 Was du heute lernst:
In Python machen wir die gleichen Dinge wie mit den Blöcken, wir müssen sie nur aufschreiben:
* Statt dem Block `[Zeige "Hallo"]` verwenden wir `print("Hallo")`.
* Statt dem Block `[Frage "Wie heißt du?" und warte]` verwenden wir `input("Wie heißt du?")`.
* Um uns eine Antwort zu merken, speichern wir sie in einer **Variable**, z.B.: `name = input("Wie heißt du?")`.

## 📝 Deine Aufgabe

Öffne die Datei `main.py`. Du siehst dort Zeilen, die mit einer Raute (`#`) beginnen. Das sind *Kommentare*. Python liest diese nicht. Hier stehen deine Anweisungen!
