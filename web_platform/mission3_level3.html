Aufgabe

1. Importiere ganz oben random.
2. Erstelle so den geheim_code.
3. Ändere deine if/while Logik ab, dass sie nun gegen diesen geheim_code prüft.
4. Starte unser Minispiel und knacke den Code in der Konsole!
