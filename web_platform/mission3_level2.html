Mission 3 - Level 2: Zu hoch, zu niedrig

Der Tresor soll uns helfen. Dazu muss er wissen, ob unsere Zahl größer oder kleiner als der Code ist.

Zahlen aus Text machen

In Python ist "5" (Text) nicht das gleiche wie 5 (Zahl). Um Text in Zahlen umzuwandeln, setzen wir ein int()Integer, auf Deutsch: Ganzzahl. darum.

mache Zahl aus input()

In Python:

tipp = int(input("Tipp: "))

if tipp < 50:
    print("Zu niedrig!")

Aufgabe

1. Umschließe das input() aus dem vorigen Level mit int(), um eine Zahl zu bekommen.
2. Vergleiche den ipp mit dem Code (z.B. 50).
3. Nutze < und >, um "Zu niedrig" und "Zu hoch" in der Schleife per if/elif auszugeben!
