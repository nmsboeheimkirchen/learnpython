# 💣 Die Bombe entschärfen 💣

Willkommen, Agent! Deine Mission für diese Woche ist es, ein Programm zu schreiben, das über Leben und Tod entscheidet (zumindest im Terminal).

## 📝 Deine Aufgabe

Öffne die Datei `main.py` und bearbeite die Zeilen, die mit `# TODO` markiert sind.

Du musst:
1. Den Benutzer fragen, welches Kabel (Rot, Blau oder Grün) er durchschneiden möchte.
2. Das Programm so programmieren, dass es je nach Eingabe explodiert (`KABUMM!`) oder entschärft (`Bombe entschärft`) wird.
3. **Das richtige Kabel ist ROT.**

## 💡 Tipps
* Du benötigst `input()`.
* Du benötigst `if`, `elif` und `else`.
* Achte auf Einrückungen! Python ist da sehr streng.
* Groß- und Kleinschreibung bei der Eingabe (z.B. "Rot" vs "rot") kann trickreich sein. Bonus: Überlege, wie du beides abfangen kannst.

Viel Erfolg! 🕵️‍♂️
