elif (Else-If) und input()

Mit elif:Kurz für „else if“ (ansonsten wenn). Prüft eine weitere Bedingung, falls die vorherige falsch war. können wir vor dem else einen weiteren Fall prüfen.

setze kabel auf input("Welches Kabel? ")
wenn kabel = "rot" dann
zeige Text "Entschärft!"
ansonsten wenn kabel = "blau" dann
zeige Text "Nichts passiert."
ansonsten
zeige Text "KABUMM!"

Vollständiger Python-Code:

kabel = input("Welches Kabel? ")
if kabel == "rot":
    print("Entschärft!")
elif kabel == "blau":
    print("Nichts passiert.")
else:
    print("KABUMM!")

Aufgabe

Füge vor dem else-Zweig mit KABUMM! ein elif (= else if) für das blaue Kabel ein. Den Code kannst du von oben kopieren.

Erklärung

if prüft zuerst Rot. Nur wenn das falsch ist, prüft elif Blau. else übernimmt ohne eigene Bedingung alle übrigen Farben. Genau ein Zweig läuft: Rot → „Entschärft!“, Blau → „Nichts passiert.“ und der Code kann erneut gestartet werden, jede andere Farbe → „KABUMM!“.
