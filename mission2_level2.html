Falsches Kabel

Ein Agent muss auf das Unerwartete vorbereitet sein. Was passiert, wenn die if-Bedingung NICHT zutrifft?

Der Else-Zweig

Mit else:Bedeutet „ansonsten“. Wird ausgeführt, wenn das if darüber falsch war, und braucht keine eigene Bedingung. können wir einen Plan B einbauen.

ansonsten

In Python:

else:
    print("KABUMM!")

Dein Schritt

Setze kabel im Feld Python-Code auf eine beliebige Farbe außer "rot" und ergänze darunter den else-Zweig von oben.

Erklärung

Ist die if-Bedingung falsch, führt Python stattdessen den eingerückten Plan B unter else aus. Dafür funktioniert jede andere Farbe, nicht nur Blau.
