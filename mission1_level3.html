Jetzt fragt das System nach deinem Namen und begrüßt dich damit.

Eingabe & Variablen

Mit input()Wartet auf Texteingabe, die mit Enter bestätigt werden muss. stellst du eine Frage. Die Antwort speicherst du in einer VariableEin benannter Speicherplatz für Daten..

setze name auf input("Wie heißt du? ")
zeige Text "Willkommen im System," und name

Python-Code zum Kopieren:

name = input("Wie heißt du? ")
print("Willkommen im System,", name)

So geht’s: Kopiere die zwei Zeilen von oben ans Ende des Felds Python-Code. Führe den Code aus, tippe deinen Namen im Bereich Ergebnis ein und bestätige mit Enter.

Erklärung

Die erste Zeile erstellt die Variable name und weist ihr sofort die Antwort der input()-Abfrage zu. „Wie heißt du?“ ist die Frage, die das System dabei stellt. Die zweite Zeile gibt die Begrüßung zusammen mit dem gespeicherten Namen im Ergebnis aus.
