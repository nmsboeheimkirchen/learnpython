Safe-Knacker - 2:

Zahlen aus Text machen

In Python ist "5" (Text) nicht dasselbe wie 5 (Zahl). Die ersten beiden Zeilen sind bereits vorbereitet: input() speichert zuerst den Text in tipp. Danach macht int(tipp)Integer, auf Deutsch: Ganzzahl. daraus eine Zahl.

Zu hoch, zu niedrig

Du rätst den Code. Der Tresor teilt dir mit, ob dein Tipp niedriger oder höher als 50 ist. 50 ist der voreingestellte Code.

Dein Programm-Plan

Tippe auf einen Block oder fahre mit der Maus darüber, um den Python-Code zu sehen.

setze tipp auf input("Tipp: ") tipp = input("Tipp: ")
setze tipp auf die Zahl aus tipp tipp = int(tipp)
wenn tipp < 50 dann if tipp < 50:
zeige Text "Zu niedrig!" print("Zu niedrig!")
ansonsten wenn tipp > 50 dann elif tipp > 50:
zeige Text "Zu hoch!" print("Zu hoch!")

Dein Schritt

Die ersten beiden Zeilen sind schon da. Ergänze ab Zeile 3 die beiden Abfragen mit ihren eingerückten print()-Hinweisen. Führe den fertigen Code einmal mit einer Zahl unter 50 und danach noch einmal mit einer Zahl über 50 aus.

Erklärung

tipp = input(...) speichert zuerst den Text. tipp = int(tipp) macht daraus eine Zahl. Die kurze Schreibweise tipp = int(input("Tipp: ")) bedeutet dasselbe und wird ebenfalls akzeptiert. if erkennt Zahlen unter 50; elif prüft Zahlen über 50. Das Level ist geschafft, sobald du beide Hinweise durch je einen Test gesehen hast.
