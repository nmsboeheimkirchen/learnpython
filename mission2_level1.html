Variablen & Bedingungen

Zuerst weisen wir einer VariableEin kleiner Speicherplatz im Computer einen Wert mit einem einzelnen = zu. Dann prüfen wir mit if und == ob etwas stimmt.

wenn kabel = "rot" dann

In Python:

kabel = "rot"
if kabel == "rot":
    print("Entschärft!")

Erklärung

Ein einzelnes = weist der Variable einen Wert zu. == vergleicht zwei Werte. Ist der Vergleich wahr, führt Python die eingerückte Zeile unter if aus.
